# README

## クローン
`$ git clone git@github.com:Hiroyuki0210/RailsTutorial.git`

## gemのインストール
`$ bundle install`

## マイグレーション
`$ rails db:migrate`

## 初期データ投入
`$ rails db:seed`

## 実行
`$ rails server`

http://localhost:3000/ にアクセス。
